# AlfredoRest

A REST API built with Javalin. It aims to retrieve data in JSON format from Alfredo's workout database.

Currently hosted at [alfredobot.com/api](alfredobot.com/api)


## Getting Started

Download project

Enter `sbt run`

## Built With

* Scala
* Javalin
* SBT
